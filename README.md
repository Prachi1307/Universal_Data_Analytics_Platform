# 📊 Universal Data Analytics Platform

A powerful Streamlit-based data analytics application that enables users to upload datasets, perform exploratory data analysis, create interactive visualizations, train machine learning models, and generate reports—all through an intuitive web interface without writing code.

---

## 🚀 Features

### 📁 Dataset Upload
- Upload CSV files
- Upload Excel (.xlsx) files
- Upload JSON files
- Upload Parquet files

### 👀 Data Preview
- Interactive dataset preview
- Column information
- Data types
- Missing value detection
- Duplicate record detection

### 📊 Dashboard

- Key performance metrics

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/universal_data_analytics_platform.git
```

### Navigate to Project

```bash
cd universal_data_analytics_platform
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will start at:

```
http://localhost:8501
```
